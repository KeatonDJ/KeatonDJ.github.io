CprE 288 - Mars Rover

This project is an embedded systems project created for CprE 288 at ISU

This project was assigned in my first embedded systems course at ISU. We were tasked with using each component we used throughout lab to navigate a course with tall thick objects (stalagmite), short objects (boulders), missing floor tiles (craters), and finally 4 thin pylons signifying the finish zone

Project Description (found on the CprE 288 - Embedded Systems course website)
